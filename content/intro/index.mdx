nut.js is a desktop automation framework for Node.js that allows you to program your mouse and keyboard with JavaScript or TypeScript!

On this page you can find:

- tutorials
- API docs
- a plugin overview
- migration guides for major versions
- and a blog

!> If not stated otherwise all docs and tutorials are targeting the latest stable release of nut.js!

If you find any errors or want to contribute, you can find the sources for this website on [GitHub](https://github.com/nut-tree/website)!
